# SteamVR Settings Integration Credits

## Original Creator
This SteamVR settings integration mod was originally created by a community member and shared in the PSVR2Toolkit community.

## What It Does
Adds a native "PlayStation VR2" section to SteamVR's settings menu, allowing users to toggle PSVR2Toolkit features directly from within VR without needing to edit configuration files or use external applications.

## Integration
This mod has been officially integrated into the PSVR2Toolkit distribution with the creator's permission, making it easier for all users to access these convenient in-VR settings.

## Features Provided
- Enable/Disable Eyelid Estimation
- Disable Play Area (Chaperone)
- Hide PlayStation VR2 Overlay
- Disable Dialog Pop-Ups
- Disable PlayStation VR2 Sense Controllers
- Disable Eye Tracking

## Technical Implementation
- **Resource-only driver** for localization strings
- **Settings schema** that integrates with SteamVR's native settings UI
- **Localization support** (currently English, extensible to other languages)
