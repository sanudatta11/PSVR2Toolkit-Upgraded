PSVR2Toolkit SteamVR Menu Installation Guide
Step 1. Download
Download both playstation_vr2_ex.zip and settingsschema.vrsettings

Step 2. Extract playstation_vr2_ex.zip to SteamVR
Extract the contents of playstation_vr2_ex.zip to C:\\Program Files (x86)\\Steam\\steamapps\\common\\SteamVR\\drivers

This is an resource folder required for the localization of the menu

Step 3. Extract settingsschema.vrsettings to PSVR2 driver
Create a new folder called settings at C:\\Program Files (x86)\\Steam\\steamapps\\common\\PlayStation VR2 App\\SteamVR_Plug-In\\resources

and move settingsschema.vrsettings into the settings folder

ENSURE YOU FIRST CREATE THE FOLDER

Step 4. Launch SteamVR
Now you can launch steamVR and you should see new section "Playstation VR2" show up
